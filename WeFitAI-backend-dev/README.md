# WeFitAI



## 프로젝트 소개

이 프로젝트는 사용자의 신체 정보를 기반으로 개인 맞춤형 운동을 추천하고, 꾸준한 건강 관리를 돕는 헬스케어 애플리케이션입니다. 사용자는 자신의 신체 변화를 그래프로 쉽게 확인하며 동기를 부여받을 수 있습니다.

<br>

## 주요 기능

* **회원가입 및 로그인**: 안전한 사용자 계정 관리 시스템
* **신체 정보 관리**: 키, 체중, 체지방률 등 개인의 신체 정보 입력 및 수정
* **BMI 자동 계산 및 분석**: 입력된 정보를 바탕으로 BMI(체질량지수)를 자동 계산하여 제공
* **맞춤형 운동 추천**: 분석된 신체 정보에 따라 가장 적합한 운동 종목과 루틴(자세, 세트 수) 추천
* **신체 정보 히스토리**: 시간에 따른 신체 정보 변화를 직관적인 그래프로 확인
* **사용자 정보 수정**: 비밀번호, 개인 정보 등을 언제든지 수정 가능

<br>

## 주요 화면 및 기능 시나리오

### 1. 회원가입 및 로그인
- **(시나리오 1, 2)** 앱 실행 시, 아이디와 비밀번호를 입력하여 로그인할 수 있습니다.
- 계정이 없는 사용자는 회원가입 버튼을 통해 아이디, 비밀번호, 개인 인적 사항을 입력하고 가입을 완료합니다.
- 로그인 정보가 일치하지 않을 경우, "아이디 또는 비밀번호가 일치하지 않습니다"라는 알림이 표시됩니다.



### 2. 메인 화면 및 부가 기능
- **(시나리오 3)** 로그인 성공 시, 기능 접근을 위한 메인 화면으로 이동합니다.
- 메인 화면에는 **[사용자 신체정보 입력]**, **[운동 진단 받기]**, **[신체정보 히스토리]**, **[사용자 정보]** 버튼이 있습니다.
- 화면 하단에는 어느 화면에서든 메인으로 돌아올 수 있는 **[홈 버튼]** 과 사용자 정보로 이동하는 **[사용자 정보 버튼]** 이 고정되어 있습니다.

### 3. 사용자 정보 수정 및 로그아웃
- **(시나리오 4)** 하단의 사용자 정보 버튼 클릭 시, 가입 시 입력했던 개인 정보(아이디, 생년월일 등)를 확인하고 수정할 수 있습니다.
- 화면 최하단의 **[로그아웃]** 버튼을 누르면 안전하게 로그아웃되고, 초기 로그인 화면으로 돌아갑니다.

### 4. 신체 정보 입력 및 분석
- **(시나리오 5)** 메인 화면의 **[사용자 신체정보 입력]** 버튼을 통해 키, 체중, 체지방률, 허리둘레, 악력 등을 기록할 수 있습니다.
- 정보 입력 완료 시, 자동으로 계산된 **BMI 지수**와 간단한 신체 분석 결과를 확인할 수 있습니다.



### 5. 맞춤 운동 진단
- **(시나리오 6)** 메인 화면의 **[운동 진단 받기]** 버튼을 누르면, 저장된 최신 신체 정보를 바탕으로 추천 운동 리스트가 나타납니다.
- 각 운동 종목을 선택하면, 정확한 운동 자세 이미지(또는 영상)와 권장 세트 수를 확인할 수 있습니다.

### 6. 신체 정보 히스토리
- **(시나리오 7)** 메인 화면의 **[신체정보 히스토리]** 버튼을 통해 과거부터 현재까지 기록된 모든 신체 정보의 변화를 그래프로 한눈에 파악할 수 있습니다.
- 그래프를 통해 체중, 체지방률, BMI 등의 변화 추이를 직관적으로 확인할 수 있습니다.
